## ⚙ Installation

Clone the repository:

bash
git clone https://github.com/Divya8210-d/DmgInfoTech.git
cd DmgInfoTech

Install dependencies:

npm install

Run development server:

npm run dev

Open in browser:

http://localhost:3000
