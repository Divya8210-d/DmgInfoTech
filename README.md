## ⚙️ Installation

Follow these steps to run the project locally.

### 1. Clone the Repository

```bash
git clone https://github.com/Divya8210-d/DmgInfoTech.git
cd DmgInfoTech
```

### 2.Install dependencies:
```bash
npm install
```
### 3.Run development server:
```bash
npm run dev
```
### 4.Open in browser:
```bash
http://localhost:3000
```
